### 实现功能：
   #####   1.基于C/S模式的通信软件
   #####   2.使用DH密钥交换协议商定密钥
   #####   3.使用对称加密算法DES进行加密

### 使用：

         VS 2019或以上版本，选择x86模式，即可运行

1.如果使用中出现，无法建立连接，或vector报错。

2.如果报错4996，部分编译器下只是警告不影响运行，如果是error：加上下面这条代码
